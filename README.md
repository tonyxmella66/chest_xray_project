# Chest X Ray Classification Project

Get the data from here: https://www.kaggle.com/datasets/ashery/chexpert use the small version


## Carbon Metrics

The CSV files contain detailed emissions data:
- **timestamp**: When the measurement was taken
- **project_name**: Name of the operation being tracked
- **duration**: Duration of the operation in seconds
- **emissions**: CO2 emissions in kg
- **emissions_rate**: CO2 emissions rate in kg/s
- **cpu_power**: CPU power consumption in W
- **gpu_power**: GPU power consumption in W (if available)
- **ram_power**: RAM power consumption in W
- **cpu_energy**: CPU energy consumption in kWh
- **gpu_energy**: GPU energy consumption in kWh (if available)
- **ram_energy**: RAM energy consumption in kWh
- **energy_consumed**: Total energy consumed in kWh
- **country_name**: Country where the computation was performed
- **country_iso_code**: ISO code of the country
- **region**: Geographic region
- **cloud_provider**: Cloud provider (if applicable)
- **cloud_region**: Cloud region (if applicable)
- **os**: Operating system
- **python_version**: Python version
- **codecarbon_version**: CodeCarbon version
- **cpu_count**: Number of CPUs
- **cpu_model**: CPU model name
- **gpu_count**: Number of GPUs (if available)
- **gpu_model**: GPU model name (if available)
- **longitude**: Geographic longitude
- **latitude**: Geographic latitude
- **ram_total_size**: Total RAM size in GB
- **tracking_mode**: Tracking mode used
